User Activity

What files were accessed?

Parse artifacts that record user file interactions.

These artifacts reveal what files and folders the user interacted with:

  • What files were opened via shortcuts? — LNK files record target paths, timestamps, and volume info
  • What files were recently used by each application? — Jump Lists track per-app recent file access
  • What folders did the user navigate to? — ShellBags record folder access, including deleted and network paths
  • What files were deleted? — Recycle Bin $I files preserve original paths and deletion times
