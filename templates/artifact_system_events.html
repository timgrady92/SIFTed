Parse Windows Event Logs for security and system activity.

  • Who logged on and when? — Security log events 4624, 4625, 4634
  • What processes were created? — Security log event 4688 (if auditing enabled)
  • What services were installed? — System log event 7045
  • Were logs cleared? — Security log event 1102
