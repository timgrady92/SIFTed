File Carving

Recover deleted or hidden files from disk images.

Ready

When files are deleted, the data often remains on disk until overwritten. These workflows scan raw images to recover documents, photos, executables, and other files that no longer appear in the filesystem.

Open category

Memory Analysis

See what was running when the RAM was captured.

Ready

Memory captures preserve processes, network connections, and loaded code that disappear when a machine powers off. These workflows extract that volatile evidence to reveal what was happening at the moment of capture.

Open category

Artifact Triage

Build a timeline of activity from host artifacts.

Ready

Operating systems record timestamps everywhere—file access, program execution, user actions. These workflows combine those scattered timestamps into a single chronological view to reconstruct what happened and when.

Open category

Artifact Parsing

Extract evidence from specific Windows artifacts.

Ready

Windows stores forensic gold in structured files—program execution history, recently opened files, USB device records, and event logs. These workflows parse those artifacts into readable formats you can search and filter.

Open category

Metadata Extraction

Reveal hidden information embedded in files.

Ready

Files contain more than their visible content—GPS coordinates in photos, author names in documents, creation timestamps, and software fingerprints. These workflows extract that metadata to establish provenance, location, and attribution.
